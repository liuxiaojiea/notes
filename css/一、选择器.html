9.子级选择器及权重

存在包含关系的元素选择，通过父级确定选择子级
权重：所有选择器之和
语法：父级选择器 > 子级选择器 { 属性名 : 属性值}

  • 父级选择器是用来确定取值范围

  • 子级选择器才是我们要添加样式的那个元素

  • 子级选择器必须是紧邻的父子关系

9.伪类选择器

锚伪类（a的四种状态）

  • a:link {color: #FF0000} /* 未访问的链接 */默认状态

  • a:hover {color: #FF00FF} /* 鼠标移动到链接上 */鼠标经过的状态

  • a:active {color: #0000FF} /* 选定的链接 */鼠标点击时的状态

  • a:visited {color: #00FF00} /* 已访问的链接 */鼠标点击后的状态

    奇偶行变色

    隔行变色: 奇数行 2n+1 odd 偶数2n even
    li:nth-child(2n+1){background:#555}
    li:nth-child(2n){background:#555}

    第一和最后

    li:first-child{background-color: #ff1d39}
    li:last-child{background-color: #5dffbb}

匹配同类型中的第n个同级兄弟元素E
E:nth-of-type(n) { sRules }

10.伪元素

:after 标签内容之后添加一个子元素

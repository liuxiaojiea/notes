新数组方法

fill()

fill()方法的作用是使用一个固定值来替换数组中的元素。该固定值可以是字母、数字、字符串、数组等等。fill()方法还有两个可选参数，表示填充的起始位置和结束位置。

　　//用数字来替换数组中的元素。　

var arr3 = [12,23,34,45,56];

arr3.fill(123);

console.log(arr3)  // [123,123,123,123,123]

　　



　　//使用字符串来替换数组中的元素。

var arr3 = [12,23,34,45,56];

arr3.fill("hello");

console.log(arr3)  // ["hello", "hello", "hello", "hello", "hello"]

　　



　　//使用数组来替换数组中的元素。

var arr3 = [12,23,34,45,56];

arr3.fill([123]);

console.log(arr3)  // [[123], [123], [123], [123], [123]]





　　//规定替换的起始位置和结束位置。

var arr3 = [12,23,34,45,56];

arr3.fill(1234,1,3);

console.log(arr3)  // [12, 1234, 1234, 45, 56]



　//完整用法：fill(value,start,end)  其中第2、3个参数是可选的。表示替换的起始位置和结束位置（注：不包含结束位置本身所在的元素）

forEach 遍历数组

ary.forEach((item,index,ary)=>{
item 是数组中的每一项
index 是当前项对应的索引
ary 原数组
})

  • 没有返回值

map 遍历

  • 与forEach()的参数用法一样

  • 返回值不一样

var www=[1,2,3].map(()=>{

    return 123

})//=>[123,123,123]

//返回值是个数组；数组的长度跟循环的数组一样，数组的每一项是由回调函数的返回值决定

Array.from() 类数组转数组

使用条件
1、该类数组对象必须具有length属性，用于指定数组的长度。如果没有length属性，那么转换后的数组是一个空数组。
2、该类数组对象的属性名必须为数值型或字符串型的数字

filter() 过滤器

返回值是一个数组
ary.filter(()=>{return 0})
filter里面的回调函数的return后面的条件是true，就会把这一项返回给数组；如果是false就不返回了

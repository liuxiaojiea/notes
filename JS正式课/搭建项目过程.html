搭建项目过程

  1. 想在某文件夹中搭建一个项目

  2. 使用 cmd 切入到当前目录

  3. vue create 项目名

  4. 接下来选择自己的配置

  5. 敲回车之后 会自动安装 项目需要的所有依赖

1、先下载node
2、npm 安装 vue-cli3.0
npm i -g @vue/cli
yarn global add @vue/cli
3、使用脚手架搭建项目
找到对应的目录 vue create 项目名
4、搭建完成之后 npm run serve

git工作流程

gitlab github 码云
三个不同的存储代码的地方
都是使用的git命令
本地编辑
git status
git add .工作区代码提交到缓存区
git commit -m '' 缓存区提交到历史区
git push 历史区代码提交到工作区
使用提交历史，回退到制定版本
git log 查看提交记录
git reset --hard 版本号 撤回了本地的历史版本
git push -f强制提交到工作区

git分支操作

从哪个分支切出去的，那么新建的分支就和哪个分支的内容保持一致
git branch查看分支
git branch 分支名 新建分支
git merge 分支名 合并分支

常用的Linux命令

  1. mkdir 创建文件夹

  2. cd 进入文件夹（切换路径）

  3. touch 创建文件

  4. vi 打开某个文件 >按i键进行编辑 >ctrl+C > :wq 保存并退出

  5. cat +文件名 查看文件

  6. rm 删除文件

  7. ls 查看当前文件夹下的目录

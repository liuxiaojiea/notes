组件

有两种声明方式：1.function；2.class
组件名 首字母必须大写
div、h1…原有的HTML标签是react的内置组件
组件使用时，类似vue，直接当做标签名使用即可
react组件中的数据来源有两个：属性(props)和状态(state)
props里的属性不能修改，属于父组件的
state里的属性可以修改，属与该组件私有的

纯组件

纯函数：只跟形参有关，函数执行不会对外界有影响，不会有任何副作用
纯组件：展示的内容只跟props有关；组件本身不会修改除了props之外的任何数据
目的：提高复用率

事件

写法: onClick(驼峰式命名)
如果直接写onClick={this.fn};则fn执行时的this指向不可控，且不能传参；所以需要强制改变函数的this指向。
有两种方法：

  • onClick={this.fn.bind(this,1,2)}

  • onClick={(e)=>{this.fn(e,1,2)}}

用bind写法需要注意传参顺序，事件对象会放在最后一个传
阻止默认事件
原生可以通过e.preventDefault()或者return false阻止
但在react中，只能通过e.preventDefault() 阻止

数据传递-父传子

通过自定义属性传递，子组件通过this.props.xxx调用
不可以修改props里面的属性

数据传递-子传父

通过自定义属性传递一个函数体，子组件通过this.props.xxx(参数)，让函数体执行并传参

import React from 'react';

import ReactDOM from 'react-dom';

class Count extends React.Component{

    render() {

        let {count}=this.props

        return (<div className=''>

        <h1>展示模板</h1>

        <h2>当前数字是{count}</h2>

        </div>

        );

    }

}

class Button extends React.Component{

    render() {

        let {text='按钮',onClick}=this.props

        return (<div>

            <button onClick={()=>{onClick('hello world')}}>{text}</button>

        </div>

        );

    }

}

class App extends React.Component {

    constructor(props) {

        super(props);

        this.state={

            count:0,

             name:'world'

        }

    }

    add(str){

        this.setState({

            count:++this.state.count,

            name:str

        })

    }

    reduce(str){

        this.setState({

            count:--this.state.count,

            name:str

        })

    }

    render() {

        return <div className=''>

            <Count count={this.state.count}></Count>

            <Button onClick={(str)=>{this.add(str)}} text='增加'></Button>

            <Button onClick={(str)=>{this.reduce(str)}} text='减少'></Button>

            <h1>{this.state.name}</h1>

        </div>;

    }

}

ReactDOM.render(<App></App>,window.root)

setState

大部分是异步
有三种传参方式

  1. 对象:当一个函数中同时有两个setstate时，会将两个setstate合并

  2. 函数：参数是函数时不会合并

  3. 对象加函数：回调函数在render之后触发

ref父组件可获取子组件

ref的两种方式
1、直接跟字符串 通过this.refs.xxx调用元素
2、跟函数体 通过函数体中自定义的属性this.xxx调用
ref发现给的是个函数体时，会默认执行该函数体；
并且把获取到的元素，通过参数的方式传给我们自己定义的属性
